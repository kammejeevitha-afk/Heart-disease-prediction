# Heart-disease-prediction
Heart Disease Prediction uses machine learning algorithms to analyze medical data and determine the risk of heart disease in individuals. It helps in early diagnosis, enabling better treatment decisions and reducing the chances of severe health complications.
## Overview
This project uses machine learning algorithms to predict the likelihood of heart disease based on patient health data. It helps in early detection and supports better medical decision-making.
## Features
Predicts risk of heart disease using input health parameters
Uses machine learning models for accurate predictions
Provides visualization of data and model performance
Easy-to-use and customizable
## Dataset
The dataset contains medical attributes such as:
Age
Sex
Blood Pressure
Cholesterol Level
Heart Rate
Blood Sugar
Chest Pain Type
(You can use datasets like the UCI Heart Disease dataset or Kaggle datasets)
## Technologies Used
Python
Pandas
NumPy
Matplotlib / Seaborn
Scikit-learn
## Installation
Clone the repository:
git clone https://github.com/your-username/heart-disease-prediction.git
Navigate to the project folder:
cd heart-disease-prediction
Install required libraries:
pip install -r requirements.txt
## Usage
Run the main script:
python main.py
Enter the required health parameters to get prediction results.
